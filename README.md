# Hurmat Naz — Portfolio

Single-page portfolio built with React, Vite, TypeScript, Tailwind CSS, and Framer Motion.

## Setup on another PC

1. Unzip this folder
2. Open a terminal inside it
3. Run:

```bash
npm install
npm run dev
```

4. Open the local URL shown in the terminal (usually http://localhost:5173)

## Build for production

```bash
npm run build
npm run preview
```
